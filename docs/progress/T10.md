#### Quyết định kỹ thuật
- Used `reqwest` with `rustls-tls` (not native-tls) for portability in Docker slim images.
- JWKS cache uses `tokio::sync::RwLock` with time-based TTL (300s default).
- OIDC discovery fetches `/.well-known/openid-configuration` to find `jwks_uri`.
- Test RSA keys generated with `ssh-keygen -t rsa -b 2048 -m PEM`, force-committed despite `*.pem` gitignore.
- Added `aud` field to JwtClaims for proper audience validation (Keycloak puts client_id in both `aud` and `azp`).
