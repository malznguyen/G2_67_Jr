- **Trait object-safe bằng boxed future/stream**: Tránh `async_trait` dependency mới trong API và cho phép handler sau này giữ `Box<dyn LlmProvider>`.
- **`chat_stream` trả `ChatStream` provider-level**: Sprint này parse SSE thành delta; T49 sẽ bọc thành Axum SSE route/citation parser.
- **Embedding mặc định vẫn Ollama**: `ProviderConfig::from_global` dùng DeepSeek cho chat/graph và Ollama cho query embedding, đúng assumption Sprint 6A.
- **OpenAI embedding vẫn pin 768 dim**: Request `/embeddings` gửi `dimensions=768`, giữ tương thích Qdrant collection 768-dim.
