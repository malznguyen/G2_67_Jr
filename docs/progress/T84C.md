#### Key architectural decisions (locked for T85)
1. Next.js 16 App Router — hybrid SSR/CSR
2. Auth.js v5 + Keycloak (not raw keycloak-js for main flow)
3. openapi-typescript + openapi-fetch + hand-written SSE/upload
4. TanStack React Query v5 for all server state
5. Zustand minimal (tenant id, theme, sidebar only)
6. shadcn/ui + react-hook-form + zod + recharts
7. POST SSE via `@microsoft/fetch-event-source` (EventSource rejected)
8. T85 order: Foundation → Auth → Shell → Settings → ACL → Documents → Chat → Usage
